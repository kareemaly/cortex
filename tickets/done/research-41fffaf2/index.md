## Goal

Compare the Claude Code and OpenCode agent default configurations in Cortex to understand what differs between the two integrations — prompts, args, MCP setup, lifecycle hooks, and any other defaults.

## Research Questions

1. What default prompts does each agent type get? How do they differ in structure and content?
2. What default arguments and configuration are set for each?
3. What MCP tool bindings does each agent receive?
4. Are there lifecycle hooks or other defaults that exist for one but not the other?
5. What gaps exist in the OpenCode defaults compared to Claude Code?

## Acceptance Criteria

- Side-by-side comparison of defaults for both agent types documented
- Key gaps and differences clearly called out
- Findings captured in a doc for future reference
